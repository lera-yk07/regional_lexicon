# Техническое задание на проект: "Региональная лексика России"
## 1. Цель проекта
Создание веб-сервиса для сбора, анализа и сравнения уникальных слов и выражений из разных регионов России. Платформа позволяет пользователям изучать языковое разнообразие страны, добавлять региональные слова в общий словарь, сравнивать лексику между городами и просматривать статистику по использованию слов.

## 2. Роли пользователей
Гость: Может просматривать словарь, искать слова по различным критериям, сравнивать лексику городов, просматривать статистику

Зарегистрированный пользователь: Может добавлять новые региональные слова через форму с валидацией

Администратор: Полный доступ к управлению контентом через админ-панель Django, модерация добавленных слов

## 3. Модели данных
Region (Регион): Географический регион России (область, край, республика)

Поля: name (CharField, название), description (TextField, описание), created_at (DateTimeField, дата создания)

City (Город): Конкретный город в регионе

Поля: name (CharField, название), region (ForeignKey, связь с Region), population (IntegerField, население), founded_year (IntegerField, год основания), created_at (DateTimeField, дата создания)

Word (Слово): Региональное слово или выражение

Поля: word (CharField, слово), meaning (TextField, значение), example (TextField, пример использования), city (ForeignKey, связь с City), user (ForeignKey, связь с User), created_at (DateTimeField, дата добавления), views_count (IntegerField, количество просмотров)

## 4. Ключевой функционал
Пользователь просматривает каталог региональных слов с пагинацией, поиском и фильтрацией по городам

Пользователь добавляет новое слово через форму с валидацией данных на стороне сервера

Пользователь сравнивает лексику двух городов и видит общие/уникальные слова

Система отображает статистику по регионам и городам с использованием библиотеки Pandas

## 5. Внешние интеграции / Аналитика
Библиотека Pandas: Используется для анализа данных в разделе статистики. Библиотека выполняет:

Группировку данных по регионам и городам

Расчет средней и суммарной статистики просмотров

Формирование таблиц для отображения популярных слов

Анализ распределения слов по географическим объектам

Встроенная аутентификация Django: Для регистрации и входа пользователей используется стандартная система аутентификации Django с кастомизированными шаблонами.

Bootstrap 5: Используется для создания адаптивного и современного интерфейса, работающего на мобильных устройствах и десктопах.
